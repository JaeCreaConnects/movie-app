# Movie Trailer Application
# Functionalities: #
1) Spring Boot Backend Server
2) Node Frontend Server utilizing React
3) Mongodb to hold movie information
4) Review API and Movie API: The frontend makes axios calls to the backend which then makes a REST api call to our Mongodb
5) Displays all movies and reviews for each movie
6) Ability to write a review, store the review, and view the review for specific movies
7) TODO: Login/Regsiter functionality *may* come later, however, the purpose of this app was to create and display basic movie trailers and reviews.
